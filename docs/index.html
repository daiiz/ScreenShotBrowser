ScreenShot Browser for Android

Coming Soon!!

スマートフォンで撮影されたウェブページのスクリーンショットを用いて、写っているウェブページを検索するアプリです。
このアプリを使えば、
  • Twitterでシェアされた記事のスクリーンショット
  • メモとしてスクショしておいたニュース記事
  • LINEでのチャットで受け取ったAmazonの商品ページのスクリーンショット
など、URLがわからないページをさっと見つけることができます！

■ 使い方

1. アプリを起動


2. スクリーンショット写真を選択


3. OCR!

Google Cloud Vision APIのTextAnnotationを使用しています。
ボタン「ステップ2」を押した後は少々時間がかかりますのでお待ち下さい。




4. 検索に使うキーワードを選ぶ

最大で5個までのキーワードを選択できます。
探しているページの特徴をよく表しているキーワードを選んでみてください。




5. 検索

「検索する」を押して検索しましょう。
